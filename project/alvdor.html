Alvdor

Text to image with AI image generator

Year : 2022 | Role : Development, Design by : Zainal Arifin

Visit link

Bytelight

Alvdor is an image generator, which can generate images based on text in just a few moments. It is the result of Zainal Arifin's collaboration as UI Designer with me as the Web Designer.

The design of this web landing page is very modern and clean. With a striking main color combined with a dark mode background color, making this design look cool and professional. The addition of simple animations on buttons, images and icons, as well as animations when scrolling.

Tools and technology

These designs and web pages are of course designed and built using tools and technology, so they are accessible. What I use in this project are :

  • HTML & CSS/SASS, For design slicing
  • Bootstrap v5.3, I use the latest version of the bootstrap framework to make it easy for me to build fast and responsive websites.
  • AOS, Animates when scrolling down
